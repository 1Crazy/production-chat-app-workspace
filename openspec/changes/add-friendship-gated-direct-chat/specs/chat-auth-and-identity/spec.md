## MODIFIED Requirements

### Requirement: 系统必须支持基础身份资料与联系人发现规则
系统 MUST 保存用户昵称、头像、唯一业务标识和可配置的联系人发现范围，并只在授权规则内暴露可发现信息。联系人发现结果用于搜索用户、查看最小资料和发起好友申请，不得直接作为单聊放行依据。

#### Scenario: 通过业务标识搜索联系人
- **WHEN** 用户输入存在的业务标识并满足可发现规则
- **THEN** 系统返回最小必要的联系人资料摘要，允许用户发起好友申请或查看资料页状态

#### Scenario: 隐私限制联系人发现
- **WHEN** 被搜索用户关闭了对应发现入口或搜索者不满足访问条件
- **THEN** 系统不得返回超出策略允许范围的资料信息
